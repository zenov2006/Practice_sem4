# Руководство пользователя
## Поиск по лекциям университета

---

## Быстрый старт

### 1. Откройте приложение
Перейдите по адресу: http://localhost:3000

### 2. Загрузите документы
- Перетащите файлы PDF или DOCX в зону загрузки
- Или кликните по зоне и выберите файлы
- Максимальный размер файла — 20 МБ

### 3. Выполните поиск
- Введите ключевые слова в поле поиска
- Нажмите кнопку «Найти» или клавишу Enter

---

## Загрузка документов

| Действие | Результат |
|----------|-----------|
| Перетащить файл | Файл загружается и индексируется |
| Выбрать через диалог | Аналогично |
| Статусы: | |
| «Загрузка...» | Файл передаётся на сервер |
| «Готово (X чанков)» | Файл проиндексирован, можно искать |

---

## Поиск

| Элемент | Описание |
|---------|----------|
| Поле ввода | Введите поисковый запрос |
| Кнопка «Найти» | Запускает поиск |
| Карточки результатов | Показывают найденные фрагменты |

Пример запроса:
- `лекция` — найдёт все документы с этим словом
- `Elasticsearch поиск` — найдёт по нескольким словам

---

## История поиска

- Все запросы сохраняются автоматически
- Нажмите на запрос в истории, чтобы повторить поиск
- Кнопка «Очистить историю» удаляет все запросы

---

## Технические требования

- Современный браузер (Chrome, Firefox, Edge, Safari)
- Docker Desktop (для запуска)

---

## Частые вопросы

Вопрос: Файл не загружается
Ответ: Проверьте, что формат PDF или DOCX, размер до 20 МБ

Вопрос: Поиск не даёт результатов
Ответ: Убедитесь, что файлы загружены и проиндексированы (статус «Готово»)
